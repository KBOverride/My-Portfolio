Swear Be Gone (Chrome Extension)

Includes Design, HTML, Front End Development

Demo

Click here to install the Google Chrome Extension and see it in use.

Overview

When I first started using JavaScript, I thought of creating a Chrome Extension to become familiar with it. My Chrome extension takes swear/bad words and replaces them with asterisks(*) across websites. I also want to give out a quick reminder that the extension only works for text in html pages and not other elements such as images, sound, etc.
