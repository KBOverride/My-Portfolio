FriendsFM Python Program

Includes Design, Google API

Demo

Unfortunately, a live demo takes too much time and effort to process (about 15 minute wait). However, I have provided song samples and requests can be made too.

Overview

The python program takes lyrics from a song the user picks and tries to match each lyric to a word said on the popular sitcom, "Friends". By having access to the subtitles, I can search through the parts of the subtitles for the keywords(lyrics). From there, I can manipulate the audio using the timestamps and take each part of the audio (usually 1-6 seconds) and send them through the Google Speech API. The results show me the exact timing of the word being said thus creating an audio file containing it. Finally, I concatenate all the audio files to create the final song.

Programs and API's

  • Python
  • pysrt (Subtitle Parser)
  • PyLyrics (Retrieves song lyrics from lyrics.wikia.com)
  • pydub (Manipulating audio)
  • Google Speech API
