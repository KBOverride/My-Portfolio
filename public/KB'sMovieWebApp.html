Overview

I wanted to further develop my skills using an API so I decided to build a useful web application that shows which movies and in theatres and coming soon. The web application is based on Material Lite Design that does not rely on JavaScript framworks but insteads aims to optimize for cross-device use. The design makes it so that the user can have very fast loading times on pages.
